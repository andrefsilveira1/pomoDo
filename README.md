# PomoDo

### Motivação:

Por ter dificuldade em concentrar em tarefas do cotidiano, como estudar e trabalhar, eu preciso utilizar técnicas que aumentem minha produtividade e que me permitam ficar mais tempo concentrado e focado. Assim, criei um pequeno sistema para que me ajudasse a me avisar quando eu deveria parar para iniciar um pequeno intervalo.
